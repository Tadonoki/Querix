# Querix

Querix adalah platform belajar SQL interaktif berbahasa Indonesia yang dirancang untuk pemula, calon Data Analyst, mahasiswa, dan career switcher.

Aplikasi ini membantu pengguna belajar SQL dari nol melalui materi terstruktur, latihan interaktif, playground SQL, dan tantangan berbasis kasus sederhana.

## Fitur Utama

- Materi SQL terstruktur dari dasar
- Tantangan SQL yang selaras dengan materi
- Playground SQL untuk latihan bebas
- Tampilan hasil query dalam bentuk tabel
- Progress belajar
- Streak latihan berbasis hari
- Login dan daftar frontend-only
- UI modern, bersih, dan beginner-friendly
- Full Bahasa Indonesia

## Materi yang Dipelajari

- Apa itu SQL
- SELECT
- Memilih kolom
- WHERE
- ORDER BY
- LIMIT
- DISTINCT
- COUNT
- SUM
- AVG
- GROUP BY
- HAVING
- JOIN dasar
- CASE WHEN
- Contoh kasus Data Analyst sederhana

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Monaco Editor
- Recharts
- LocalStorage untuk data demo/progress

## Status Project

Project ini masih berupa frontend MVP.  
Backend, database, autentikasi asli, dan eksekusi SQL sungguhan akan dikembangkan pada tahap berikutnya.

## Tujuan Project

Querix dibuat sebagai project pembelajaran dan portfolio untuk membangun platform edukasi SQL yang mudah dipahami oleh pengguna Indonesia.
